# pfgo 数据获取TG Bot

## 功能:

-  当前只实现 /all 查询全部规则使用情况

## 部署

### polling部署
- pip install -r requirement.txt
- 填写config.ini文件
- python polling.py -c config.ini 运行即可

### webhook部署
- pip install -r requirement.txt
- 填写config.ini文件
- python polling.py -c config.ini 运行即可

## 使用

将机器人拉入群中通过 /help 获取使用命令  
ps: 群管理员必须有机器人创建者否则会拒绝查询
